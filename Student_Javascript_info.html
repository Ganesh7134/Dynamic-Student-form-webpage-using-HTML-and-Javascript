JavaScript Code Explanation

URL Parameter Handling

const urlParams = new URLSearchParams(window.location.search);

  • Creates a new URLSearchParams object
  • Parses the query string from the current URL (everything after the ?)
  • Example: For student_db.html?Fname=John&Lname=Doe, it extracts Fname=John&Lname=Doe

Displaying Form Data

document.getElementById('displayFname').textContent = urlParams.get('Fname');

  • Gets the 'Fname' parameter from the URL
  • Sets it as text content for the element with ID 'displayFname'
  • Example: Sets "John" if URL contains ?Fname=John

document.getElementById('displayLname').textContent = urlParams.get('Lname');

  • Same as above but for last name
  • Retrieves 'Lname' parameter value

document.getElementById('displaySID').textContent = urlParams.get('SIN');

  • Gets student ID from 'SIN' parameter
  • Displays in element with ID 'displaySID'

document.getElementById('displayDOB').textContent = urlParams.get('dob');

  • Retrieves date of birth from 'dob' parameter
  • Shows in 'displayDOB' element

Handling Radio Buttons (Gender)

const gender = urlParams.get('gender') || 'Not selected';

  • Gets 'gender' parameter value
  • Uses logical OR (||) to provide default "Not selected" if no value exists
  • Example: Returns "Male" if URL has gender=Male, otherwise "Not selected"

document.getElementById('displayGender').textContent = gender;

  • Displays the retrieved gender value
  • Shows either the selected gender or default text

Handling Checkboxes (Hobbies)

const hobbies = urlParams.getAll('hobbies').join(', ');

  • getAll() gets all values for 'hobbies' parameter (since multiple can be selected)
  • join(', ') combines them into a comma-separated string
  • Example: Converts ["Sports","Music"] to "Sports, Music"

document.getElementById('displayHobbies').textContent = hobbies || 'None';

  • Displays the joined hobbies string
  • Shows "None" if no hobbies were selected (empty string is falsy)

Remaining Fields

document.getElementById('displayEmail').textContent = urlParams.get('email');

  • Gets and displays email address
  • Shows empty string if no email provided

document.getElementById('displayPassword').textContent = urlParams.get('password');

  • Displays password (note: insecure in URL)
  • In production, should never expose passwords like this

document.getElementById('displayDesc').textContent = urlParams.get('description');

  • Shows the self-description text
  • Empty if no description provided

document.getElementById('displaySkills').textContent = urlParams.get('skills');

  • Displays selected skill from dropdown
  • Empty if no selection made

Key Notes

Security Warning:

  • GET method exposes all data in URL (including passwords)
  • Never use for sensitive information in production

Form Requirements:

  • Form must use method="get"
  • All inputs need proper name attributes
  • Checkboxes need same name but different values
  • Radio buttons need same name to work as a group

Default Handling:

  • Only gender and hobbies have default fallbacks
  • Other fields will show empty string if not provided

Data Types:

  • get() returns single values (text inputs, radios, selects)
  • getAll() returns arrays (checkboxes)
